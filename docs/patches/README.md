# Patch Documentation

This directory contains documentation for patches applied to upstream dependencies.

## Purpose

Each patch documentation file explains:
- Why the patch is needed
- What changes it makes
- How to verify the patch works
- How to update patches when upstream changes

## Files

- `picamera2.md` - Documentation for Picamera2 library patches

## Patch Management

Patches are stored in `../patches/` directory and applied automatically by setup scripts. See main documentation for details.
