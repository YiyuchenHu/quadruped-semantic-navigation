# Documentation

This directory contains comprehensive documentation for the quadruped semantic navigation project.

## Purpose

Documentation covers:
- Project overview and architecture
- Quick start guides
- Hardware setup instructions
- Software stack and dependencies
- Troubleshooting guides
- Patch documentation

## Structure

- `00-overview.md` - Project overview and architecture
- `01-quickstart.md` - Installation and quick start guide
- `02-hardware.md` - Hardware requirements and setup
- `03-software-stack.md` - Software dependencies and versions
- `04-troubleshooting.md` - Common issues and solutions
- `patches/` - Documentation for upstream patches

## Reading Order

1. Start with `00-overview.md` for project understanding
2. Follow `01-quickstart.md` for setup
3. Refer to other docs as needed
