# Demo Applications

This directory contains runnable demo applications that demonstrate various features of the quadruped navigation system.

## Purpose

Demos provide working examples and test cases for:
- Object detection (YOLOv5)
- Camera integration
- Real-time processing
- Integration testing

## Structure

Each demo is in its own subdirectory with:
- Demo script(s)
- README with usage instructions
- Required model files (in `models/` subdirectory)
- Test assets (in `assets/` subdirectory)

## Running Demos

See individual demo READMEs for specific instructions. General pattern:

```bash
cd demos/<demo_name>
python <demo_script>.py
```
